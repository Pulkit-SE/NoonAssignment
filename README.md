This repo is created for NOON's assignment.

This repo has a food delivery app code using React Native with TypeScript. 

Features - 
1. Home Screen
2. Cart Screen
3. Confirmation Screen
4. Search Screen
5. Details Screen

How to run -
1. Clone the repo.
2. run 'npm install'
3. cd ios/ && pod install
4. npm run android or npm run ios
